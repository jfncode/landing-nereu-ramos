# 🏥 Landing Page — Hospital Nereu Ramos

Landing page institucional do **Hospital Nereu Ramos (HNR)**, de Florianópolis/SC — referência estadual em infectologia e pneumologia, parte da rede pública de saúde (SUS) gerida pela Secretaria de Estado da Saúde de Santa Catarina.

## ✨ Recursos

- **Página única responsiva** (HTML + CSS + JS puro, sem dependências/build).
- Paleta institucional: **branco, azul e vermelho** (cores do hospital/SC).
- Seções: Hero, Estatísticas, História, Linha do tempo, Especialidades, Prêmios, Galeria e Contato (com mapa).
- **"Nereu Responde"** — chatbot de FAQ com avatar, 100% offline (sem API/custo), que responde dúvidas do público sobre especialidades, agendamento, horários, endereço, prêmios e mais.
- Logo oficial integrado (header, footer, hero e favicon).
- Links para matérias e fontes oficiais (SES/SC, ND Mais, AMIB/Epimed).

## 🚀 Como usar

Basta abrir o arquivo `index.html` no navegador — não há etapa de build.

```bash
# ou servir localmente
npx serve .
```

## 📁 Estrutura

```
.
├── index.html        # Página completa (HTML + CSS + JS inline)
├── DOSSIE-HOSPITAL-NEREU-RAMOS.md  # Pesquisa/dados que embasaram a página
└── fotos/            # Imagens (fachada, logo, avatar do Nereu, etc.)
```

- Fotos atuais: **SECOM / SES-SC**
